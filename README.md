# Diving Cat's 제주 다이빙 스팟 지도

제주도 프리다이빙 & 스노쿨링 명소를 한눈에 볼 수 있는 인터랙티브 지도입니다.

👉 **[지도 보러가기](https://elain2.github.io/jeju-freediving-spot/)**

![Diving Cat](free_diving_cat.png)

## 주요 기능

- **12개 다이빙/스노쿨링 스팟** 정보 제공
- **고양이 발자국 마커**로 스팟 위치 표시
  - 🔵 파란색: 프리다이빙 전문
  - 🟢 초록색: 스노쿨링 추천
  - 🟣 보라색: 둘 다 가능
- **필터 기능**: 프리다이빙 / 스노쿨링 / 초보자용 / 보트 다이빙
- **상세 정보**: 수심, 난이도, 접근성, 특이점
- **인스타그램 연동**: 각 스팟의 해시태그 검색 링크

## 수록된 스팟

| 스팟 | 지역 | 유형 | 수심 |
|------|------|------|------|
| 문섬 (한개창) | 서귀포 | 프리다이빙 | 5m~25m |
| 범섬 | 서귀포 | 프리다이빙 | 10m~30m |
| 섶섬 | 서귀포 | 프리다이빙 | 5m~20m |
| 협재 해변 | 한림 | 스노쿨링 | 1m~5m |
| 함덕 해변 | 조천 | 스노쿨링 | 2m~8m |
| 월정리 해변 | 구좌 | 스노쿨링 | 1m~4m |
| 김녕 세기알 | 구좌 | 스노쿨링 | 0.5m~3m |
| 황우지 선녀탕 | 서귀포 | 둘 다 | 2m~6m |
| 우도 서빈백사 | 우도 | 스노쿨링 | 1m~4m |
| 중문 색달 (도리빨) | 서귀포 | 둘 다 | 2m~15m |
| 월령 | 한림 | 프리다이빙 | 3m~12m |
| 이호테우 해변 | 제주시 | 스노쿨링 | 1m~5m |

## 기술 스택

- HTML/CSS/JavaScript
- [Leaflet.js](https://leafletjs.com/) - 인터랙티브 지도
- [CARTO](https://carto.com/) - 지도 타일
